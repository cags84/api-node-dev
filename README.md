# Api Node Rest

## Clone the repository

```bash
> git clone https://github.com/cags84/api-node-dev
```

## Create file .env, add next lines

```
SERVER_HOST="10.100.100.100"
SERVER_PORT=3000

# Config MondoDB
DB_CONNECTION_STRING=""
DATABASE_PROTOCOL=""
DATABASE_URL=""
DATABASE_USERNAME=""
DATABASE_PASSWORD=""
```

## Install application

```bash
> npm install
```

## Run server

```bash
> npm run dev
```
